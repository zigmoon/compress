== Installation ==

= Automatic Installation =
This process is safe and easy to install the plugin on your WordPress installation. First, login to your WordPress dashboard, then go to the <strong>Plugin</strong> menu then click on the <strong>Add New</strong> option.

Here plugin list page will open, on this page in the search box write the name of the plugin <strong>compress-analytics</strong> and press the <strong>Search</strong> button. After this, you can find our plugin name at the top of the list now just click on the <strong>Install</strong> button.

<strong>Admin Dashboard > Plugins > Add New > </strong>Search for <strong>compress-analytics</strong> plugin.

= Manual Installation =
In the manual installation, you have to download our plugin first. Extract the .zip and put the whole contents in the <strong>wp-content/plugins/</strong> directory of your WordPress installation and then activate the plugin from the <strong>Plugins page</strong>.

After activating the plugin, go to <strong>Setting > compress-analytics</strong>.
